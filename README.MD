# Pinsights v2 вЂ” Personal Aesthetic Intelligence System

> Not what you like. **Who you are.**

Pinsights is a multi-agent RAG system that analyses your saved fashion images and produces: an **Aesthetic DNA profile**, a precise **Gap Analysis** between aspiration and execution, and **grounded cultural context** вЂ” all specific, all reasoned, none generic.

## Architecture

```
Images в†’ CLIP Embedder в†’ HDBSCAN Clusterer в†’ Qdrant RAG в†’ Groq LLM
                                                    в†“
                               Intake в†’ Identity в†’ Gap в†’ Narrator
                                         (LangGraph pipeline)
                                                    в†“
                                        FastAPI (SSE streaming)
                                                    в†“
                                        React/Vite/D3 frontend
```

### The Four Agents

| Agent | Function |
|-------|----------|
| **Intake** | CLIP embeds images в†’ HDBSCAN clusters в†’ palette extraction |
| **Identity** | Queries `fashion_history` Qdrant collection в†’ names aesthetic worlds with Groq |
| **Gap** | Queries `aesthetic_execution` Qdrant collection в†’ surfaces specific execution gaps |
| **Narrator** | Synthesises all outputs в†’ structured JSON with cultural context statements |

### RAG Collections

| Collection | Chunks | Purpose |
|------------|--------|---------|
| `fashion_history` | 60 | Cultural origin, era, visual codes, fabric language for 15+ aesthetics |
| `aesthetic_execution` | 40 | Specific execution criteria, common misses, actionable tells |

## Setup

### Prerequisites
- Python 3.11+
- Node.js 18+
- Docker Desktop (for Qdrant)
- Groq API key

### 1. Start Qdrant

```bash
docker compose up -d
```

### 2. Python environment

```bash
cd backend
pip install -r requirements.txt
```

### 3. Environment variables

```bash
cp .env.example .env
# Edit .env and fill in:
#   GROQ_API_KEY=your_key
#   UNSPLASH_ACCESS_KEY=your_key (optional)
```

### 4. Ingest the RAG corpus

```bash
python -m backend.rag.ingest_corpus
```

This embeds the 100 fashion history and execution chunks into Qdrant. Run once.
Use `--force` to re-ingest after editing corpus JSON files.

### 5. Start the backend

```bash
uvicorn backend.api.main:app --reload --port 8000
```

### 6. Start the frontend

```bash
cd frontend
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

## Data

### Preset boards

Pre-existing boards in `data/pinterest_img/`:
- `minimal/` вЂ” Minimalism and quiet luxury
- `rock/` вЂ” Rock and edge aesthetics
- `streetwear/` вЂ” Streetwear
- `summer/` вЂ” Summer light
- `winter/` вЂ” Winter layers

### Upload your own

Drag and drop any collection of images (min. 5, max. 50). JPG, PNG, WebP.

## API Reference

```
POST /analyze                    вЂ” Start analysis (multipart form)
GET  /analyze/{id}/stream        вЂ” SSE progress stream в†’ final result
GET  /analyze/{id}               вЂ” Get stored result
GET  /images/presets             вЂ” List preset boards
GET  /history/sessions           вЂ” List past sessions
GET  /history/drift/{a}/{b}      вЂ” Compare two sessions
GET  /health                     вЂ” Health check
```

Full interactive docs at [http://localhost:8000/docs](http://localhost:8000/docs)
